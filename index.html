メタマスクウォレットをダウンロードして利用を始める

公開日：2026年4月25日 最終更新：2026年4月25日 執筆：佐藤 健一 所要時間：約10分

メタマスク（MetaMask）は、ブラウザ拡張機能とモバイルアプリで利用できる代表的な暗号資産ウォレットで、Web3サービスやNFTマーケットプレイスへの接続にも使われます。本記事では、メタマスクのダウンロード方法、ブラウザ拡張機能とモバイルアプリそれぞれのインストール手順、初回起動時のウォレット作成と初期設定、シードフレーズの安全な保管、そして日常的なメタマスク ログインの流れまでを、初めての方でも迷わないように一つずつ順を追って解説します。最初に押さえてほしいのは、ダウンロードは必ず公式サイト metamask.io と公式ストアから行うという一点です。検索結果や広告に表示される偽サイトを避けるための具体的な見分け方も後半で取り上げます。

この記事の要点

メタマスク ダウンロードは 公式サイト metamask.io、Chrome ウェブストア、App Store / Google Play の三経路に限定する。シードフレーズは画面に表示された瞬間からオフラインでのみ管理する。日常のメタマスク ログインはアプリまたは拡張機能のローカルパスワードで行い、シードフレーズの再入力は復元時のみ必要になる。

メタマスクとは何か、なぜ必要なのか？

メタマスクは、イーサリアム互換のブロックチェーン上で資産を管理し、分散型アプリケーション（dApps）に接続するためのソフトウェアウォレットです。秘密鍵はユーザーの端末にのみ保存され、運営者が預かることはありません。

メタマスクは2016年にConsenSys関連の開発チームによって公開されて以来、ブラウザでイーサリアムやEVM互換チェーンを利用する際の事実上の標準ウォレットとして広く採用されてきました。本人確認やアカウント開設は不要で、インストールしてシードフレーズを生成した瞬間にウォレットが利用可能になります。一方で、運営者が秘密鍵を保管しないということは、ユーザー自身が鍵の管理責任を負うという意味でもあります。本記事では、この自己管理型ウォレット特有の責任を踏まえ、最初の一歩を安全に踏み出すための実務手順を整理します。

カストディアル取引所との違い

国内の暗号資産取引所が「カストディアル（運営者が秘密鍵を預かる）」であるのに対し、メタマスクは「ノンカストディアル」です。取引所のパスワードを忘れても本人確認で再発行できますが、メタマスクの場合はシードフレーズを失うと誰にも復元できません。逆に言えば、運営側からの凍結や引き出し制限が原則として発生しないという特性があります。両者は競合ではなく役割が異なるため、取引所からメタマスクへの送金、メタマスクからdAppsへの接続という二段構えで使われるのが一般的です。

メタマスクはどこからダウンロードすべきか？

公式入手経路は次の三つだけです。metamask.io、Chrome ウェブストア／Firefox Add-ons／Edge Add-ons、そしてApp Store／Google Play。それ以外のリンク、SNSのDM、検索広告に紛れた類似ドメインからは絶対にダウンロードしないでください。

ブラウザ拡張機能をインストールする手順は？

ブラウザ拡張機能版はパソコンでのdApps利用に最適です。Chrome／Brave／Firefox／Edge のいずれでも、所要時間は3〜5分です。

  1. 公式サイトにアクセスアドレスバーに metamask.io と直接入力します。検索結果ではなくURLを直接入力するのが最も安全です。
  2. 「Download」ボタンを選択利用しているブラウザが自動検出され、対応するストアへのリンクが表示されます。
  3. ストアで「追加」または「インストール」Chromeの場合は「Chromeに追加」、Firefoxの場合は「Firefoxへ追加」を選択します。権限の確認ダイアログが表示されたら内容を読み「拡張機能を追加」を押します。
  4. ピン留め（任意だが推奨）ブラウザ右上のパズルアイコンから MetaMask の横にあるピンマークを押すと、ツールバーに常時表示されます。誤って類似ロゴをクリックしないために、追加直後にピン留めしてください。
  5. 初回起動追加完了画面または狐のアイコンをクリックすると、ウォレット作成のセットアップ画面に遷移します。次の章に進みます。
拡張機能の正規性チェック ストアページに記載されたユーザー数、評価件数、開発元名（MetaMask）を必ず確認してください。同名の偽拡張機能がストアに紛れている事例が過去に報告されています。

モバイルアプリをインストールする手順は？

外出先でも残高確認や送金を行いたい場合、モバイルアプリは欠かせません。拡張機能と同じシードフレーズで同期できるため、両方をインストールしておくと利便性が高まります。

iOS（iPhone / iPad）

  1. App Storeを開くホーム画面のApp Storeアプリを起動します。
  2. 「MetaMask」で検索検索欄に「MetaMask」と入力し、提供元が ConsenSys Software Inc. となっているアプリを選びます。
  3. 入手をタップFace ID／Touch ID／Apple IDのパスワードで認証してダウンロードを開始します。
  4. アプリを起動ホーム画面に追加された狐アイコンをタップして初期セットアップに進みます。

Android

  1. Google Playを開くPlayストアを起動します。
  2. 「MetaMask」で検索提供元 ConsenSys Software Inc. のアプリを選択します。
  3. 「インストール」をタップ必要な権限の確認後、自動的にダウンロードとインストールが進みます。
  4. アプリを起動「開く」ボタン、またはランチャーから狐アイコンをタップして初期化を開始します。
APKファイルの直接インストールは推奨しません AndroidではAPKを外部から取得してインストールすることも技術的には可能ですが、改ざんされたバイナリを掴むリスクがあるためGoogle Play経由を強く推奨します。

ウォレット作成と初期設定の流れは？

初回起動では「新規ウォレットを作成」と「既存ウォレットをインポート」の選択画面が表示されます。新しく始める場合は前者を選びます。

  1. 「新しいウォレットを作成」を選択既存のシードフレーズを所有していない場合はこちらを選びます。
  2. 利用規約・プライバシーポリシーを確認本文を読んだうえで同意します。改善のための匿名データ送信の可否はあとから設定で変更できます。
  3. パスワードを設定これは端末ローカルでウォレットを暗号化するためのパスワードで、メタマスク ログイン時に毎回求められます。シードフレーズとは別物であり、忘れた場合はシードフレーズで再復元できます。8文字以上、推測されにくい英数字記号の組み合わせを使ってください。
  4. シードフレーズを表示12個の英単語が画面に表示されます。この時点で表示された単語列が、あなたのウォレットそのものです。
  5. シードフレーズを書き写す必ず紙にペンで書き写し、順序を維持したまま物理的に保管します。スクリーンショット、クラウドストレージ、メールへの貼り付けは厳禁です。
  6. 確認テストシードフレーズを正しく記録したかどうか、画面の指示に従って一部を再入力する確認テストが行われます。
  7. セットアップ完了ウォレットアドレス（0xから始まる42文字）が割り当てられ、残高ゼロのウォレットが利用可能になります。

シードフレーズはどう保管すべきか？

シードフレーズはあなたの資産そのものへのマスターキーであり、これを失う＝資産を失うことを意味します。同時に、誰かに見られた瞬間に資産を盗まれることも意味します。

推奨される保管方法

  • 耐火金庫の中に紙で保管する。
  • 金属プレート（クリプトスチールなど）に刻印して水・火災・経年劣化に備える。
  • 同じフレーズを物理的に離れた2か所に分散して保管する（火災・盗難に対する冗長化）。

絶対にやってはいけない保管方法

禁止事項
  • iCloud / Googleドライブ / Dropbox / Notion / Evernote / メモアプリへの保存
  • メール下書き・自分宛てメッセージ・LINE Keepへの貼り付け
  • スマホでスクリーンショットを撮ること
  • 誰かに口頭、メッセージ、ビデオ通話で伝えること（公式サポートを名乗る人物含め、誰であっても）

サポートを装う詐欺の典型パターン

SNSやDiscord、Telegramで「公式サポート」を名乗るアカウントから声をかけられ、「同期のためにシードフレーズを入力してください」と誘導されるケースは2024〜2026年にかけて急増しています。正規のメタマスク サポートはシードフレーズを尋ねません。例外なく詐欺と判断してください。

メタマスク ログインはどう行うのか？

メタマスク ログインとは、初回設定時に決めたローカルパスワードを入力して、暗号化されたウォレットを端末上で復号する操作のことです。シードフレーズを毎回入力する必要はありません。

拡張機能でのログイン

  1. ブラウザ右上の狐アイコンをクリックします。
  2. パスワード入力欄が表示されたら、初期設定で決めたパスワードを入力します。
  3. 「ロック解除」を押すと、残高画面に遷移します。

モバイルアプリでのログイン

  1. ホーム画面から狐アイコンをタップしてアプリを開きます。
  2. パスワードを入力するか、設定で生体認証を有効にしている場合は Face ID／指紋認証で解除します。
  3. 残高画面が表示されればログイン完了です。

パスワードを忘れた場合は？

パスワードはシードフレーズを使えばリセットできます。アプリまたは拡張機能の「ウォレットをリセット／復元」を選び、12単語のシードフレーズを正しい順序で入力してから新しいパスワードを設定してください。シードフレーズも紛失している場合は、残念ながら復元手段はありません。これがノンカストディアルウォレットの原則です。

安全に利用するための注意点は何か？

ウォレットそのものを安全に作成しても、接続先のサイトや署名する取引内容を確認しないと資産は守れません。最終防衛線は常にユーザー自身です。

  • 接続先のドメインを毎回確認する。署名画面に表示されるサイトのドメインがブックマークしている公式と一致しているかを確認します。
  • 署名する内容を理解する。「Approve」「setApprovalForAll」など、無制限の引き出し権限を与える署名はとくに慎重に扱います。不要な許可は revoke.cash などで定期的に取り消します。
  • 大きな金額にはハードウェアウォレットを併用する。Ledger や Trezor をメタマスクと連携させると、署名を物理デバイスのボタンで承認する仕組みになり、PCがマルウェアに感染しても秘密鍵は外部に漏れません。
  • ブラウザ拡張機能の追加には特に慎重に。悪意のある拡張機能はクリップボードを書き換えて送金先アドレスをすり替えることがあります。新しい拡張機能を入れる前に評価・レビュー・開発元を必ず確認します。
  • OSとブラウザを最新に保つ。ゼロデイ脆弱性経由の攻撃を避けるため、自動更新を有効にしておきます。

よくある質問

メタマスクの利用は無料ですか？

メタマスクのインストールおよびウォレット作成自体は無料です。ブロックチェーン上で送金や取引を行う際にネットワーク手数料（ガス代）が発生し、メタマスク内蔵のスワップ機能を利用する場合は別途サービス手数料が上乗せされます。これらは利用前に画面で確認できます。

拡張機能とモバイルアプリは同じシードフレーズで同期できますか？

同じシードフレーズで「インポート」することで同期できます。一方の端末で生成したシードフレーズを、もう一方の端末で「既存ウォレットをインポート」から入力すれば、同一のアドレスと残高にアクセスできます。アプリ間の自動同期ではなく、決定論的な鍵生成によって同じアドレスが導出される仕組みです。

シードフレーズを画面で見せれば家族と共有しても大丈夫ですか？

共有相手の信頼度に関わらず、画面で見せた瞬間にスクリーンショットや写真撮影のリスクが生じます。共有が必要な相続・贈与の場面では、紙に書いたフレーズを耐火金庫または信頼できる弁護士・税理士のもとで保管する物理的な手段を選んでください。

メタマスク ログインができなくなった場合の最終手段は？

パスワードを忘れた場合はシードフレーズで復元、シードフレーズも失った場合は復元手段がありません。これは欠陥ではなく、ノンカストディアルウォレットの根本仕様です。だからこそ、シードフレーズの物理保管は最初の数分で必ず完結させてください。

複数のアカウントを1つのメタマスクで管理できますか？

同一シードフレーズから派生する複数のアカウント（アドレス）を「アカウント追加」から作成できます。用途別（保管用、取引用、テストネット用など）に分けると整理しやすくなります。シードフレーズが1つでもアドレスは複数持てるのが特徴です。

取引所のアプリとメタマスクの違いは何ですか？

取引所は運営者が秘密鍵を管理する「カストディアル」、メタマスクは利用者本人が管理する「ノンカストディアル」です。取引所は法定通貨との売買や本人確認による復元が可能で、メタマスクはdAppsへの接続や自己責任での自由度が強みです。両者は競合ではなく、目的に応じて使い分けるのが標準的です。

  • お問い合わせ時：メールアドレス、お名前、ご連絡内容など、利用者が任意に提供される情報。
  • アクセス情報：IPアドレス、ブラウザ種別、リファラ、閲覧ページ、滞在時間など、ブラウザが自動送信する情報。
  • Cookie：本サイトはアクセス解析のために匿名化されたCookieを使用する場合があります。

本サイトは秘密鍵、シードフレーズ、ウォレットアドレスを一切収集しません。これらの情報を求めるフォームは本サイトには存在しません。

3. 利用目的

  • お問い合わせへの返信
  • 本サイトの品質改善およびユーザー体験の向上
  • 不正アクセス・スパムの検知および防止
  • 法令または正当な要請に基づく対応

4. 第三者提供

当編集部は法令に基づく場合を除き、利用者の同意なく個人情報を第三者に提供しません。

5. 外部サービスとアクセス解析

本サイトはアクセス解析や配信最適化のため、第三者の解析ツールを利用する場合があります。これらのツールは匿名化された統計情報を取得するもので、個人を特定する情報を含みません。利用者はブラウザ設定でCookieを無効化できます。

6. データ保管期間

お問い合わせメールは原則として返信完了後12か月間保管し、その後速やかに削除します。アクセスログは原則として最大26か月間保管します。

7. 利用者の権利

利用者は当編集部に対して、自己の個人情報の開示・訂正・削除・利用停止を請求できます。請求は metapaint@gmail.com までご連絡ください。本人確認のうえ、合理的な期間内に対応します。

8. セキュリティ

当編集部は個人情報への不正アクセス、紛失、改ざんを防止するため、技術的および組織的に合理的な安全管理措置を講じます。本サイトはHTTPSによる暗号化通信を提供します。

9. 子どものプライバシー

本サイトは13歳未満の子どもから個人情報を意図的に収集しません。誤って収集が判明した場合は速やかに削除します。

10. ポリシーの変更

本ポリシーは必要に応じて改定される場合があり、改定時は本サイトに掲示します。重要な変更については別途お知らせします。

11. お問い合わせ窓口

個人情報の取り扱いおよび本ポリシーに関するご質問は metapaint@gmail.com までお寄せください。
